📋 System Overview

Project Description

Drawing Modal is a sophisticated, serverless drawing canvas application that captures comprehensive interaction data with 13+ measurements per point. It provides real-time UI updates, session tracking, hover path detection, and complete JSON export of all drawing behavior metrics.

Technology Stack

  • Frontend: Vanilla JavaScript ES6+, HTML5, CSS3
  • Canvas: 2D Context for rendering
  • Input: Pointer Events API (unified input)
  • Data Format: JSON export, PNG image export
  • Architecture: 100% client-side, serverless

Altitude & Azimuth Calculations

Altitude: Angle of stylus from surface (0-90°)

Azimuth: Compass direction of stylus tilt (0-360°)

Altitude Calculation: altitude = 90° - arctan(√(tiltX² + tiltY²)) * (180/π) If tiltX = 0 AND tiltY = 0: altitude = 90° (perpendicular) Else: altitude decreases as tilt increases Azimuth Calculation: azimuth = atan2(tiltY, tiltX) * (180/π) If azimuth < 0: azimuth += 360° (normalize to 0-360 range) Result represents direction: 0°=East, 90°=South, 180°=West, 270°=North
