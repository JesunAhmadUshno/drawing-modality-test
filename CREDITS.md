## 📚 Methodologies & Standards

### Computer Vision Algorithms
- **Hu Moments** - Shape descriptors (M. K. Hu, 1962)
  - Used for: Shape invariant feature extraction

- **SIFT (Scale-Invariant Feature Transform)** - Feature detection algorithm (David Lowe, 1999)
  - Used for: Keypoint detection and feature matching

- **SSIM (Structural Similarity Index)** - Image quality metric (Wang et al., 2004)
  - Used for: Drawing similarity comparison

- **Convex Hull** - Computational geometry algorithm
  - Used for: Shape compactness and boundary analysis

### Signal Processing
- **Velocity & Acceleration Analysis** - Kinematic analysis
  - Used for: Movement pattern detection, tremor analysis

- **Pause Detection Algorithms** - Temporal analysis
  - Used for: Hesitation patterns, behavioral markers
